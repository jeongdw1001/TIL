## 3) 클라우드
- Iass(아쓰) : infrastructure
- Pass(파쓰) : platform
- Sass(사쓰) : software

  - 가상화기술(Virtualization)
    - 메모리 가상화(자원 가상화)
    - 자원 분할
      - OS,CPU,storage, memory, network, ...
    - 자원 통합 (클러스터링)
      - CPU, storage, memory, network
- scale out(<->in) : 서버를 여러대로 만들어서 묶어둠
- scale up(<->down) : cpu, memory, 성능 업그레이드
- 필요할 때 늘리고 필요없으면 줄이기 가능(확장성, 탄력적)
