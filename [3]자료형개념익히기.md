1) 조건문 : 만일에 ~하다면 이라는 조건에 대한 명령을 구현할 때 사용한다.

```python    
[단일 형식] 
    if 조건: 
        True 실행문 
    else : 
        False 실행문

[다중 형식]
   if  조건:
      True  실행문
   elif 조건2:
       True  실행문
   elif 조건3:
       True  실행문
   else :
      False 실행문 
```

